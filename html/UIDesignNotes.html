    • Design Roles
      • User Researcher
        • Understand user behaviors and needs.
        • Gather and analyze user data.
        • Conduyct surveys, interviews, and usability testing
      • UX Designer
        • Responsible for the overall feel of the product.
        • Focus on how users will interact with the product.
        • Use wireframing tools like Balsamiq or prototyping tools like Figma or Adobe XD
      • UI Designer
        • Focus on designing the user interface of a site.
        • Create detailed mockups developers use to build the final product.
      • Padding sweet spot is 1.3em-1.9em
      • Line height should be around 1.5
      • Think of every element as having a series of rows and columns.
      • Add equal whitespace in container.
      • WCAG 2.5 Contrast Guidlines (minimum AA) say the visual presentation of text and images of text has a contrast ration of at least 4.5:1, except for large text which should have a contrast ratio of at least 3:1.
      • WCAG 2.5 Contrast Guidlines (enhanced AAA) say the visual presentation of text and images of text has a contrast ration of at least 7:1, except for large text which should have a contrast ratio of at least 4.5:1.
      • The size of every UI element must be carefully considered .

    Design Patterns
    Commonly used solutions to common problems in software design.
    Dark Mode
    Dark mode is a color scheme that uses light-colored text, icons, and graphical elements on a dark background.
    Card Layouts
    Card layouts are a design pattern that groups information into cards that can be easily rearranged and reorganized.
    Modal Dialogs
    Modal dialogs are a type of dialog that appears on top of the main content and requires the user to interact with it before they can continue.
    Shopping Carts
    Shopping carts are a design pattern that allows users to add items to a cart and then proceed to checkout.
    Breadcrumbs
    Breadcrumbs are a design pattern that shows the user's current location within a website or application.
    User Testing
    User testing includes A/B testing, heatmaps, and user behavior analytics.
    White Space
    The empty space between the elements in your user interface.
    Alignment
    Alignment is the process of ensuring that every element is positioned correctly in relation to other elements.
    Contrast
    Contrast is being in a strikingly different state from something else.
